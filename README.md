# SecureAuth

A secure authentication system built with Next.js, TypeScript, Prisma, and PostgreSQL. This project focuses on implementing practical authentication and web application security features using modern development practices.

## Features

- User Registration
- User Login
- JWT Authentication
- HttpOnly Cookie Authentication
- Password Hashing (bcrypt)
- Protected Routes
- Account Lockout
- Login Rate Limiting
- CSRF Protection
- Security Headers
- Audit Logging
- Input Validation

## Tech Stack

- Next.js
- TypeScript
- React
- Tailwind CSS
- Prisma ORM
- PostgreSQL
- JWT
- bcrypt
- Zod
